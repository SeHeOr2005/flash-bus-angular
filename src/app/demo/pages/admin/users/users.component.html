Pasos para asignarlo:

  1. Abre Postman o un navegador
  2. Llama a GET http://localhost:8080/roles → copia el id del rol ADMINISTRADOR_SISTEMA
  3. Tu userId está en el JWT (lo puedes ver en localStorage → currentUser → id)
  4. Llama a POST http://localhost:8080/user-role/user/{userId}/role/{roleId}
  5. Cierra sesión e inicia sesión nuevamente
